# Free Music Equalizer

A real-time audio visualizer and equalizer built with Python, PyQt5, and PyAudio.

## Features

- Real-time audio spectrum visualization
- Colorful gradient bars (blue to red)
- FFT-based frequency analysis
- ~30 FPS refresh rate

## Requirements

- Python 3.11+
- See `requirements.txt` for dependencies

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python AudioEqualizer.py
```
